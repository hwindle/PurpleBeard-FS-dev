Higher Order Functions in JavaScript

What are higher order functions?

Functions that take in other functions as arguments, and/or functions that return functions.

array.forEach

This changes the original array you are working on, in other words it mutates state.

This could be used when changing data types in place, for example changing strings into
numbers or breaking up long numbers into digits.

Map

One key difference between map and forEach is that map returns a new array and
leaves the original array.
Unlike array.forEach, you can also chain map and other JavaScript methods together, using a dot.
It doesn't mutate state, changing the original array can sometimes lead to bugs.
Map could be used with an array of DOM (document object model) elements from a web page.
Each element could then have styles or HTML attributes applied to it and it would be easier to write that way.

Filter

Used to validate and check each element of an array, by using an if condition
without having to write an if statement. Useful for unchecked user inputs or
parsing other data from an unknown source. This could also be used to check
arrays of objects for properties or for whether that object is an instance of a certain class.

Reduce

Useful for summing an entire array or performing a mathematical
operation and accumulating that result. Reduces the array to one value.
